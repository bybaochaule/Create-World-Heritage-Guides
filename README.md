# Create World Heritage Guides

Create professional educational guides and interpretive resources about UNESCO World Heritage Sites for teachers, travel educators, students, museums, documentary creators, and public-history teams.

This Codex skill helps produce accurate, classroom-ready and audience-aware materials such as site profiles, historical context, cultural and natural significance summaries, preservation notes, map context, timelines, key facts, glossary terms, quiz questions, answer keys, discussion prompts, classroom activities, documentary outlines, museum labels, and heritage education resources.

## What It Creates

- UNESCO World Heritage Site profiles and fact sheets
- Full educational guides for classrooms or public programs
- Historical, cultural, geological, and ecological context
- Preservation and stewardship notes
- Map context and site-location explanations
- Timelines, key facts, and glossary terms
- Quiz questions, answer keys, and discussion prompts
- Classroom activities and field-study handouts
- Museum labels and exhibit text
- Documentary outlines, narration beats, and interview prompts
